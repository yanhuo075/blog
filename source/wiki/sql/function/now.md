## SQL NOW() 函数

------

NOW() 函数返回当前系统的日期和时间。

### SQL NOW() 语法

```
SELECT NOW() FROM table_name;
```

## SQL NOW() 实例

下面的 SQL 语句从 "Products" 表中选取产品名称以及当天的价格：

示例

```
SELECT ProductName, Price, Now() AS PerDate
FROM Products;
```
