## SQL LOWER()函数

------

SQL lower()字母大小写转换函数，将字母转成小写 - 返回根据当前字符集映射所有字符改变为小写，即返回小写的字符串。

### LOWER(str)

返回根据当前字符集映射所有字符改变为小写，即返回小写的字符串。

```
SQL> SELECT LOWER('ngrok'); +---------------------------------------------------------+ | LOWER('ngrok') | +---------------------------------------------------------+ | ngrok | +---------------------------------------------------------+ 1 row in set (0.00 sec)
```
