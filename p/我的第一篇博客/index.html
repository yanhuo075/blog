OpenSSL升级版本

安装依赖包

yum update -y
yum install -y perl-IPC-Cmd perl-Pod-Html
yum install -y openssl openssl-devel
yum install -y make zlib zlib-devel openssl-devel sqlite-devel bzip2-devel libffi libffi-devel gcc gcc-c++
yum install -y wget

查看当前openssl版本

openssl version

备份原文件

mv /usr/bin/openssl /usr/bin/openssl.bak

下载指定openssl版本源码包

wget https://www.openssl.org/source/openssl-1.1.1w.tar.gz

解压源码包并进入指定目录

tar -xzvf openssl-1.1.1w.tar.gz && cd openssl-1.1.1w

编译源码安装

./config -Wl,-rpath=/usr/lib64 --prefix=/usr/local/openssl --openssldir=/usr/local/openssl --libdir=/usr/lib64
make -j 4 && make install

查看版本修改是否生效

openssl version -a

备注：

openssl升级后dnf安装软件包会报错，所以不建议升级openssl版本
